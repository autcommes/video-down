# 应用图标说明

本目录包含应用程序的图标文件，用于不同平台和尺寸。

## 所需图标文件

为了支持跨平台构建，需要以下图标文件：

### Windows
- `icon.ico` - Windows 图标文件（已存在）
  - 应包含多个尺寸：16x16, 32x32, 48x48, 64x64, 128x128, 256x256

### macOS
- `icon.icns` - macOS 图标文件
  - 应包含多个尺寸：16x16, 32x32, 64x64, 128x128, 256x256, 512x512, 1024x1024

### Linux/通用
- `32x32.png` - 32x32 像素 PNG 图标
- `128x128.png` - 128x128 像素 PNG 图标
- `128x128@2x.png` - 256x256 像素 PNG 图标（高 DPI）
- `icon.png` - 512x512 像素 PNG 图标（推荐）

## 图标设计建议

1. **简洁明了**：图标应该清晰表达应用功能（视频下载）
2. **可识别性**：在小尺寸下也能清晰识别
3. **品牌一致性**：所有平台使用相同的设计风格
4. **高对比度**：确保在不同背景下都清晰可见

## 图标生成工具

可以使用以下工具生成多尺寸图标：

- **在线工具**：
  - https://icon.kitchen/ - 免费的应用图标生成器
  - https://www.appicon.co/ - 多平台图标生成

- **命令行工具**：
  - ImageMagick - 批量转换图标尺寸
  - iconutil (macOS) - 生成 .icns 文件

## 创建图标的步骤

1. 设计一个 1024x1024 的主图标（PNG 格式，透明背景）
2. 使用图标生成工具创建所有所需尺寸
3. 将生成的文件放置在此目录
4. 确保 tauri.conf.json 中的图标路径正确配置
